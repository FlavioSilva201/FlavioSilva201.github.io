Unidades de Medida CSS

Unidades Absolutas

25cm -> Centimetros

25mm -> Milimetros

25in -> inches (1in = 96px = 2.54cm)

25px -> pixels (1px = 1/96th of 1in)

25pt -> points (1pt = 1/72 of 1in)

25pc -> picas (1pc = 12 pt)

Unidades Relativas

50em -> Relative to the font-size of the element (2em means 2 times the size of the current font)

50ex -> Relative to the x-height of the current font (rarely used)

50ch -> Relative to width of the "0" (zero)

50rem -> Relative to font-size of the root element

50vw-> Relative to 1% of the width of the viewport*

50vh -> Relative to 1% of the height of the viewport*

50vmin -> Relative to 1% of viewport's* smaller dimension

50vmax-> Relative to 1% of viewport's* larger dimension

50% -> Relative to the parent element
