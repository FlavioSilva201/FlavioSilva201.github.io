Position

Atributo Descrição Exemplo
Absolute O elemento é removido do fluxo do documento e outros elementos se comportam como se não estivessem lá, enquanto todas as outras propriedades posicionais trabalhariam nele. Absolute
Fixed O elemento é removido do fluxo do documento como elementos absolutamente posicionados. Na verdade, eles se comportam quase da mesma forma, apenas os elementos de posição fixa são sempre relativos ao documento, não a um pai em particular, e não são afetados pela rolagem. Fixed
Relative A posição original de um elemento permanece no fluxo do documento, assim como o staticvalor. Mas agora right / left / top / bottom / z-index funcionará. As propriedades posicionais "cutucam" o elemento da posição original nessa direção. Relative
Static Todo elemento tem uma posição estática por padrão, portanto, ele permanecerá no fluxo normal da página. Portanto, se houver um conjunto right / left / top / bottom / z-index , não haverá efeito nesse elemento. Static
Sticky (experimental) O elemento é tratado como um relativevalor até que o local de rolagem da viewport atinja um limite especificado; nesse momento, o elemento assume uma fixedposição em que é solicitado que ele permaneça. Sticky
