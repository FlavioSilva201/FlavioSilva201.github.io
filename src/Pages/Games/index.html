  • Invasores do Espaço

    This is a game inspired by the 1978 original "Space Invaders" by Taito.
    This game was written in Visual Basic 2010, with the objective of being one of the many game on my secondary final project, where I created some game modes, like "Team Deathmatch" for 2 or 4 players.
    Another game mode is "Points" where up to 6 players must defeat the invaders and the one with the most points wins.
